# Changelog


## [1.1.0] - 2024-11-20

### Added
- 添加基于 JWT 的身份验证机制
- 添加 authMiddleware 中间件用于验证 token
- 添加图片缓存控制
- 添加登录状态检查
- 添加统一的认证请求处理函数 fetchWithAuth

### Changed
- 移除全局 IMyselfToken 变量
- 修改登录接口，返回 JWT token
- 修改所有需要认证的接口使用 authMiddleware
- 改进重新上传功能，优化图片更新机制
- 改进错误处理机制

### Fixed
- 修复图片重复上传不更新的问题
- 修复图片缓存导致的显示问题
- 修复登录状态失效的处理

### Security
- 加强了接口的访问控制
- 改进了用户会话管理
- 添加了 token 过期机制

## [1.0.0]

### Added
- 初始版本发布
- 基础的图片上传功能
- 图片管理界面
- 图片预览功能
- 图片删除功能
- 简单的登录验证
